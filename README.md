# Restaurant Review Project

This project involves working with a restaurant review domain using SQLAlchemy. The project has three models: `Restaurant`, `Review`, and `Customer`. The relationships between these models are defined to represent a restaurant review system.

## Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/restaurant-review-project.git

Creating virtual environment

python -m venv venv
source venv/bin/activate

Installing dependencies
pip install -r requirements.txt

Running and debugging

python debug.py

Usage
The models.py file contains the SQLAlchemy models for Restaurant, Review, and Customer. You can interact with these models using SQLAlchemy queries. The debug.py script provides a basic setup for testing and debugging.
